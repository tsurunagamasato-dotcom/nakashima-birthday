Google スプレッドシートと連携するための手順です

1

Google スプレッドシートを作成する

新しいスプレッドシートを作成し、1行目に以下のヘッダーを入力してください。

氏名	所属部署	誕生日
田中 一郎	製造部	1985-06-15
鈴木 花子	品質管理部	1990-11-03

※ 誕生日は YYYY-MM-DD 形式（例: 1990-06-15）で入力してください

2

Google Apps Script を設定する

スプレッドシートのメニュー「拡張機能」→「Apps Script」を開き、以下のコードを貼り付けて保存してください。

const SHEET_NAME = 'Sheet1'; // シート名を確認して変更

function doGet(e) {
  const sheet = SpreadsheetApp.getActiveSpreadsheet()
                  .getSheetByName(SHEET_NAME);
  const rows  = sheet.getDataRange().getValues();
  const headers = rows[0];
  const data = rows.slice(1)
    .filter(r => r[0])
    .map(r => ({
      name:  r[0],
      dept:  r[1],
      bday:  r[2] instanceof Date
               ? Utilities.formatDate(r[2], 'Asia/Tokyo', 'yyyy-MM-dd')
               : String(r[2])
    }));
  return ContentService
    .createTextOutput(JSON.stringify({ ok: true, data }))
    .setMimeType(ContentService.MimeType.JSON);
}

function doPost(e) {
  const payload = JSON.parse(e.postData.contents);
  const sheet = SpreadsheetApp.getActiveSpreadsheet()
                  .getSheetByName(SHEET_NAME);
  if (payload.action === 'add') {
    sheet.appendRow([payload.name, payload.dept, payload.bday]);
  }
  if (payload.action === 'delete') {
    const rows = sheet.getDataRange().getValues();
    for (let i = rows.length - 1; i >= 1; i--) {
      if (rows[i][0] === payload.name && rows[i][2] === payload.bday) {
        sheet.deleteRow(i + 1); break;
      }
    }
  }
  return ContentService
    .createTextOutput(JSON.stringify({ ok: true }))
    .setMimeType(ContentService.MimeType.JSON);
}
3

Web アプリとしてデプロイする

Apps Script の画面で「デプロイ」→「新しいデプロイ」→「種類の選択：ウェブアプリ」を選択します。
「アクセスできるユーザー」を 全員 または 組織内全員 に設定して「デプロイ」を押してください。

4

Web App URL を貼り付ける

デプロイ後に発行された URL（https://script.google.com/macros/s/.../exec）を「マスタ管理」画面の GAS URL 欄に貼り付けて「保存」→「同期」を押してください。

5

SharePoint ポータルに掲載する

このHTMLファイルを SharePoint のドキュメントライブラリにアップロードし、そのリンクをポータルページに貼り付けてください。
または「埋め込み」ウェブパーツを使って iFrame として組み込むことも可能です。
